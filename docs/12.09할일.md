### 2. Emergency Agent 풀버전 테스트

**테스트 시나리오:**

1. **재난 정보 조회**
   ```
   "서울 재난문자 확인해줘"
   → 최근 7일/30일/6개월 재난문자 조회
   ```

2. **날씨 및 옷차림**
   ```
   "부산 날씨 알려줘"
   → 현재 날씨, 체감온도, 옷차림 추천
   ```

3. **여행 안전도 평가**
   ```
   "강릉 여행 가도 돼?"
   → 종합 위험도 점수 (0-10)
   → 날씨 + 재난 + 시설 접근성 종합 평가
   ```

4. **응급 상황 대응**
   ```
   "홍대에서 다쳤어"
   → 가장 가까운 병원, 경로 안내, 119 안내
   ```
